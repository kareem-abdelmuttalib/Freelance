## Project Description
SAR centralizes student activities: registration, booking, history, and calendar visualization. The interface uses glassmorphism, gradients, and smooth animations to deliver a modern, university-inspired experience.

## Page Explanations
- **Home:** Hero with CTA, live highlights, categories, and about section plus footer contacts.
- **Registration:** Validated form with inline errors and success state; semantic fields for student data.
- **Booking:** Booking form + resources sidebar; confirm CTA triggers client-side confirmation.
- **History:** Responsive gallery of past events; modal popups with details and imagery.
- **Calendar:** JavaScript calendar with month/week/year toggles and sample events rendered per view.
- **Contact:** Contact form validation, university details, and embedded campus map.
